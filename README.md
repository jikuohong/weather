⛅️ Cloudflare Weather Bot (Dual-Engine Edition)基于 Cloudflare Workers 部署的 Telegram 天气预报机器人。采用 和风天气 (QWeather) 作为地理编码与基础气象主力，彩云天气 (Caiyun) 作为高精度分钟级降雨预警备用。🌟 核心功能全地域查询：发送 /weather [地名]（如：/weather 杭州）自动获取目标位置预报。默认地守望：直接发送 /weather 获取温州鹿城（可自定义）的详细天气。双引擎驱动：和风天气：负责地名转坐标、实时温度、体感、湿度及风力。彩云天气：负责 24 小时逐小时降雨深度分析，计算精准的起止时间。自动降雨预警：支持 Cron Triggers 定时巡检，一旦预测有雨，自动推送到 Telegram（带 KV 去重，一场雨仅提醒一次）。Web 预览：支持通过浏览器直接访问 Worker 域名查看当前预报。🛠 环境变量配置 (Environment Variables)在 Cloudflare Workers 控制台的 Settings -> Variables 中添加以下变量：变量名描述来源TG_BOT_TOKENTelegram 机器人 Token@BotFatherTG_CHAT_ID你的 Telegram 数字 ID@userinfobotCAIYUN_TOKEN彩云天气 API 令牌彩云科技开放平台QWEATHER_KEY和风天气 API Key和风天气控制台注意：和风天气 Key 需在后台开通 “GeoAPI” 和 “天气预报” 权限。🚀 部署指南新建 Worker：在 Cloudflare 创建一个新的 Worker。粘贴代码：将 index.js 中的代码全部覆盖到编辑器中。绑定 KV：创建一个名为 WEATHER_KV 的 KV 命名空间并绑定。设置 Webhook：在浏览器访问以下链接（替换为你自己的 Token 和域名）：https://api.telegram.org/bot[你的TOKEN]/setWebhook?url=https://[你的域名]/配置定时器：在 Triggers 选项卡中添加 Cron Trigger，推荐 0 * * * *（每小时整点运行）。📖 使用指令/weather - 获取默认位置（温州鹿城）的实时天气和降雨分析。/weather [地名] - 获取指定城市的实时预报。自动预警 - 机器人会在检测到未来两小时有雨时主动发信。📄 开源协议MIT License. 仅供学习与折腾使用。
